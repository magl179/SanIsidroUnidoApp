Redes Sociales Asociadas Las Redes Sociales Asociadas te permiten que puedas iniciar sesión por medio de tu cuenta de Facebook o Google, si deseas desconectar una cuenta social asegurate primero de tener una contraseña activa, caso contrario cambia tu contraseña para que puedas iniciar sesión por medio de un formulario Dispositivos Asociados Los dispositivos asociados se registran cuando inicias sesión para permitir que recibas notificaciones en tu celular, si desconectas un dispositivo ya no podrás recibir notificaciones en dicho dispositivo No puedo visualizar algunas opciones en la aplicación Si observas algún problema con la aplicación, cierra y vuelve a abrir la aplicación o cierra sesión y vuelve a iniciar sesión para comprobar si el problema persiste
